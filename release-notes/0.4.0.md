- Notes now render Markdown styling inline: **bold**, _italic_, ~~strikethrough~~, # headings, and [links](url).
- Bulleted lists (`- item`) and todo checkboxes (`- [ ] todo`, `- [x] done`).
- Tap a checkbox to toggle it; checked items get strikethrough automatically.
- Pressing Return on a list line continues the list; Return on an empty bullet exits the list.
- Hardware-keyboard shortcuts: ⌘B / ⌘I / ⌘⇧X / ⌘K wrap the selection with the corresponding syntax.
- Fixed an editing bug where the cursor was jumping in long notes.
