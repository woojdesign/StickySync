- **Calmer Markdown.** Bold, italic, strikethrough, heading, and link markers now truly hide (zero width, not just dim) when the cursor isn't on their line — the word reads continuously. Tap back into the line and the markers reappear so you can edit them.
- **`[]` shortcut for checkboxes.** Type `[]` at the start of a line and it auto-expands to a proper checkbox. The file stays portable Markdown (`- [ ] item`), but you only type two characters.
