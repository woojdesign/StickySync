- Added live Markdown rendering — headings, lists, and inline styles format as you type.
- Added checkbox support: ☐/☑ glyphs render inline, and clicking a checkbox toggles it.
- Markdown markers now hide as you type and reveal when the cursor enters the line, with checkbox markers expanding while you edit them.
- Added automatic list continuation and paragraph indentation for list items, and fixed a cursor-at-end editing bug.
- Added sticky sharing — share a note with someone and accept shared stickies.
- Restored the note's "Share text… / Share with someone…" menu, with shares now created through the system share picker.
- Sharing now exposes all four permission options.
- Recipients who don't have StickySync now arrive on a Wooj landing page through the shared link instead of a dead end.
- Opening a shared sticky now presents it as a deliberate arrival moment.
